Patient Data Manager
The Patient Data Manager is a tool used to search, edit, and create FHIR resources tied to a given patient. The interface allows for easy navigation and guided resource creation. Users don’t need to be FHIR standard experts since the tool takes care of the FHIR knowledge for them. The app requires SMART OAuth2 access to patient data through an EHR or test Sandbox.

Tasks include:
  • Viewing FHIR resources tied to a patient
  • Creating and removing FHIR resource instances
  • Exporting FHIR resources in JSON format
Designed for
Clinicians, Resource Developers, & Patients
FHIR Compatibility
DSTU 3
